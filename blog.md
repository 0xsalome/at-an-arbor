# Obsidian互換マークダウン記法ガイド

このブログで使用できるObsidian互換のマークダウン記法です。

---

## 見出し

```markdown
## h2見出し（大きめ）
### h3見出し（中程度）
#### h4見出し（小さめ）
```

---

## リスト

```markdown
- 箇条書き
- ビュレットポイント付き

1. 番号付きリスト
2. 自動で番号表示
```

---

## コールアウト（注意書きボックス）

Obsidianと同じ書き方で、色付きのボックスが表示されます。

```markdown
> [!note] メモ
> 補足情報を書く

> [!tip] ヒント
> 便利な情報

> [!warning] 注意
> 気をつけること

> [!important] 重要
> 必ず読んでほしいこと

> [!quote] 引用
> 誰かの言葉
```

| タイプ | 色 | アイコン |
|--------|-----|---------|
| note | 青 | 📝 |
| tip | 緑 | 💡 |
| warning | 黄 | ⚠️ |
| important | 赤 | 🔥 |
| quote | 紫 | 💬 |

---

## ハイライト

```markdown
==強調したいテキスト==
```

黄色いマーカーで強調表示されます。

---

## タスクリスト

```markdown
- [ ] 未完了のタスク
- [x] 完了したタスク
```

チェックボックス付きで表示されます。

---

## 引用

```markdown
> 引用文
> 複数行もOK
```

左側にボーダー、背景にグラデーションが付きます。

---

## テーブル

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
| D   | E   | F   |
```

ホバー時にハイライトされます。

## 記事の更新（Regrown機能）

記事を更新する場合：

1. frontmatterの`updated`を更新日時に変更
2. 更新内容と初版の間に`---`を**2つ連続**で書く

```yaml
---
title: 記事タイトル
date: 2026-02-01
updated: 2026-02-11
type: blog
tags:
  - blog
---
更新後の内容をここに書く...

---
---

初版の内容（元の記事）...
```

### 自動で起こること

- ヘッダーに「᚛ Regrown」バッジが表示
- 「᚛ Log」セクションで更新履歴を表示
- 二重線の位置に「᚛ 初版 / 日付」ヘッダーが自動挿入
- ウィスパーナビに「更新しました」とアナウンス追加

---

## Frontmatter

```yaml
---
title: 記事タイトル
date: 2026-02-01
updated: 2026-02-11 07:25  # 更新時に追加
type: blog
tags:
  - blog
  - essay  # エッセイの場合
unlisted: true  # 非公開記事の場合
---
```

| フィールド | 説明 |
|-----------|------|
| title | 記事タイトル |
| date | 公開日（YYYY-MM-DD） |
| updated | 更新日時（省略可） |
| type | `blog` 固定 |
| tags | タグ（blog, essay など） |
| unlisted | trueで非公開（直リンクのみアクセス可） |
